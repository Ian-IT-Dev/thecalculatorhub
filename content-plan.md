# Content Plan — 12-week keyword calendar

This calendar targets low-competition, evergreen long-tail queries that funnel into existing calculators. One piece per week. Track each in Google Search Console after publishing; once a piece earns steady impressions, write 1–2 follow-up pieces targeting adjacent queries.

## Cadence
- **New calculator** every 2nd week (engine is generic — copy an existing calculator page, swap form fields, add one `Calculators.<key>()` function in `js/main.js`).
- **Article/explainer** on alternating weeks — 600–1,500 words, uses `content-template.html`, internally links to a relevant calculator.
- Submit each published URL in Google Search Console → URL inspection → Request indexing.

## Premise
Avoid head terms ("compound interest calculator") directly — you'll never outrank Bankrate/NerdWallet. Target the **queries around** those terms: "how does compounding work if I save $200/month for 30 years", "ROI on $5000 over 3 years", etc. Long-tail first, mid-tail after you have 20+ pages indexed.

## Keyword research workflow
1. **Search Console → Performance** once you have 200+ impressions. Sort by CTR; build around what's already working.
2. **AnswerThePublic** (free tier) or **Google autocomplete** for adjacent queries.
3. **"People also ask"** boxes are gold — answer each one as an H2 inside the closest article.
4. **Aim for zero-difficulty** long tail first (specific phrases with numbers, like "$500/month for 20 years").

## Content rules of thumb
- Always include a worked numeric example.
- Always embed at least one calculator link with **pre-filled inputs in the URL** (uses the shareable-URL feature built earlier).
- Always include a FAQ JSON-LD section targeting the "People also ask" questions.
- Avoid stock-photo fluff — handwritten SVG diagrams or just clear prose ranks fine.
- Disclosure: affiliate links always down the page, never the lead of the piece.

## Success metric
By end of week 12 you should have **26 pages** indexed (14 originals + 12 new), with a few hundred to a couple thousand monthly visitors depending on luck and content quality. That's the realistic threshold to apply for AdSense. Apply in week 13 if traffic is meaningful.
